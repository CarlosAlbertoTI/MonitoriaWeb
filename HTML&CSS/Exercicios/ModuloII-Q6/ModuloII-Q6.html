A vantagem de usar o css exteno é por que o código como um todo fica mais organizado, pois separa-se html do css, uma desvantagem é que, como arquivo externo tem que ser carregado e isso pode ser ruim dependendo da conexão

No caso de usar o css interno, a vantagem é a organização pois o html e css estão separados,mas eles tambem estão na mesma pagina o que dependendo do caso pode se bom para carregamento da pagina. Mas em compensação o arquivo pode ficar muito grande

No caso do inline, a vantagem é pois vc pode criar expecificações css para cada elemento, mas a desvantagem e que o código como todo fica muito bagunçado
